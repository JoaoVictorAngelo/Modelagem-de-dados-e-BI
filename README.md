# Modelagem-de-dados-e-BI
Projeto de Modelagem e Análise de Dados com foco em Business Intelligence (BI). Inclui modelagem conceitual, lógica e física em SQL, integração e tratamento de dados, além do desenvolvimento de dashboards interativos no Power BI. O objetivo é transformar dados brutos em insights estratégicos e KPIs para apoiar a tomada de decisão.
